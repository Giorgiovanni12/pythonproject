Il progetto ha lo scopo di trascrivere in formato txt un file wav e creandone un riassunto.
Per scaricare un video qualunque da youtube si può usare {https://www.freemake.com/it/free_video_downloader_fast/}.
Invece per convertirlo in formato wav si può usare {https://convertio.co/}.

ISTRUZIONI:
Avviare il file Trascrittore.py e inserire il path in cui è presente il file wav.
Alla fine del programma verrà generato un file di tipo txt con la trascrizione del file.
Avviare il file OpenAi.py e inserire il path del file txt, generando un riassunto del testo fornito.
